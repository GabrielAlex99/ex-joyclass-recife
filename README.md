# Exercício — Marco Zero, Recife

## Como executar
1. Extraia a pasta.
2. Abra o arquivo `index.html` no navegador, ou abra a pasta no VS Code.
3. Para testar com Live Server, clique com o botão direito em `index.html` e escolha **Open with Live Server**.

## Requisitos atendidos
- Cabeçalho (`header`), conteúdo principal (`main`) e rodapé (`footer`);
- Imagens com textos alternativos (`alt`);
- Semântica textual inline: `strong`, `em`, `mark` e `small`;
- Conteúdo dividido em `section`, `article`, `figure` e `figcaption`;
- Estilos feitos diretamente com o atributo `style` em cada elemento.
